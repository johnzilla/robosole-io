# robosole-io — RoboSole consumer/merch site

This is the **public** DTC website for RoboSole (robosole.io). Static HTML + Tailwind
(standalone binary) + vanilla JS. See [README.md](README.md) to run/build.

## Source of truth (read before changing brand-facing things)
The canonical brand/product truth lives in the **private sibling repo `../robosole`**
(the "brain", where the Cowork product effort works):
- `../robosole/BRAND.md` — identity, voice, palette, Tread + Shell, RoboKlasp
- `../robosole/PRODUCT.md` — the line, specs, fitment, pricing
- `../robosole/STRATEGY.md` — roadmap (Asimov-first → Phase 2)
- `../robosole/RELEASES.md` — drops & launch calendar

Before touching copy, product names, colors, or prices, read those. If a brand/product
decision changes, **update it in `../robosole` first**, then reflect it here. Don't invent
brand facts in this repo.

## This repo is PUBLIC — hard rules
- **Never** add strategy, costs, suppliers, unreleased plans, or anything from
  `../robosole` that isn't already meant for customers' eyes.
- Keep the voice consumer/sneaker-culture (swagger lives here). The corporate `.com` is
  where restraint goes — for now robosolehq.com just redirects here.

## Current MVP guardrails (Phase 1)
- **Asimov-first**, but keep copy platform-neutral ("humanoid robots").
- **No REK / combat / Unitree / fighter endorsements** on the site yet — that's Phase 2.
